# Exploratory Data Analysis (EDA) Project

This repository contains a Python script demonstrating **Exploratory Data Analysis (EDA)** techniques. The project focuses on analyzing and visualizing a dataset to uncover insights, detect patterns, and prepare the data for further analysis or modeling.

---

## Features

1. **Dataset Loading**
   - Loads a dataset in CSV format for analysis.

2. **Data Inspection**
   - Displays the first few rows of the dataset.
   - Provides information on the dataset's structure, column types, and size.

3. **Missing Value Detection**
   - Identifies missing values in each column to assess data quality.

4. **Descriptive Statistics**
   - Generates summary statistics for numerical columns.

5. **Data Visualization**
   - Histograms and KDE plots for numerical columns.
   - Count plots for categorical columns.
   - Correlation matrix with a heatmap for numerical features.

6. **Outlier Detection**
   - Uses boxplots to identify outliers in numerical data.
